# Portfolio Website (Vite + React)

This is a personal portfolio website built with Vite and React.

## Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Customize with Your Resume

All content is centralized in `src/portfolioData.js`.

Replace placeholder values with your own details from your resume:

- `profile`: name, title, summary, location, contact details
- `about`: short professional story
- `experience`: role history and bullet highlights
- `projects`: project descriptions, stack, links
- `skills`: grouped skills
- `education`: degree and school history
- `contact`: social links and resume URL

Once updated, your portfolio page will reflect those changes automatically.
